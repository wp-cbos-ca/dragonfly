The procedure is almost the same for Vimeo, select "Share" (which is the paper airplane at the top right of the video when you hover over it), then "Embed". You can adjust the settings there by choosing "Show Options" and then adjusting these below the video shown. Then copy and paste the code into the text you are editing. The underlying code for Vimeo would look something like this:
